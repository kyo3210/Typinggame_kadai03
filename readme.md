## 2.制作LPの説明（40文字程度）
小学生で認知度の高い「POPタイピング」風のゲームなら
配列とJSONを両方使えると思い参考にしました

## 3.工夫した点・こだわった点 
・雲イメージの文字や飛行機等、POPタイピングとは違うデザイン
・キー入力が成功したときに文字を消すようにした
・レベル1と2と分けて問題の配列を二種類作った
・キー入力が成功したときに音を鳴らすようにして心地よさをプラス
・過去スコアとハイスコアを表示
