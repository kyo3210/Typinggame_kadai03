配列とJSONを使うということで、小学生なら誰でも知っている
「ポップタイピングゲーム」風のゲームなら両方使えると思った

#3.工夫した点・こだわった点 
・雲イメージの文字
・キー入力が成功したときに文字を消すようにした
・レベル1と2と分けて問題の配列を二種類作った
・キー入力が成功したときに音を鳴らすようにして心地よさをプラス
・ゲーム中は15秒に一度飛行機が飛ぶ
・過去スコアとハイスコアを表示
